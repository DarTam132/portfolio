Guess my number

This is my first game made with the help of an udemy course.

This has a very basic functionality and simple rules. You basicly have to guess a number between 1 and 20, if you number is too high the game will tell you that your guess is too high, the same for too low. You have 20 chances to guess the correct number. As your guesses are wrong the score will decrease with 1. When your score is correct the layout will change, the high score will register and for a new round you'll have to press the again buttons witch will reset all the parameters, besides the highscore.

Languages used: HTML, CSS, JavaScript

Pig game

Here we have a multiplayer game. The rules are simple but the game is very nice. The game starts with player 1 round. The player has to push the roll dice button and the dice will generate a random score between 1 and 6, you can hold that value or you can roll again. If the dice come out as 1, the player will lose the current gained score and the turn will move to the other player. First player to react 50 point win.

Languages used: HTML, CSS, JavaScript
